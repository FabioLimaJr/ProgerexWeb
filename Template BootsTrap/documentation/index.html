Introduction

Litam Admin is a popular open source WebApp template for admin dashboards and control panels. It is built with Sass. Sass compiler makes it easier to code and customize. If you are unfamiliar with Bootstrap or Sass, visit their website and read through the documentation. All of Bootstrap components have been modified to fit the style of Litam Admin and provide a consistent look throughout the template. It is a responsive HTML template that is based on the CSS framework Bootstrap It utilizes all of the Bootstrap components in its design and re-styles many commonly used plugins to create a consistent design that can be used as a user interface for backend applications. Litam Admin is based on a modular design, which allows it to be easily customized and built upon. This documentation will guide you through installing the template and exploring the various components that are bundled with the template.

Installation

You need to install package files/dependencies for this project if you want to customize it. To do this, you must have Sass installed in your computer.Installation guide of the node can be found here. If you have installed them, just go to the root folder and run the following command in your command prompt or terminal.

Install SASS

Download the latest version of SASS and install it: https://sass-lang.com/

Run sass code in your project enters the following command in your command prompt or terminal.

sass --watch scss\style.scss:style.css

Layout (HTML)
The layout consists of four major parts:
  • ecaps-page-wrapper: A div that wraps the whole site.
  • Main Header: Contains the logo and navbar.
  • Sidebar: Contains the user panel and sidebar menu.
  • Content: Contains the page header and content.
